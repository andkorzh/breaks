# Фамиклоны

Тут будет большая таблица by @eugene-nes, со списком всех неофициальных ревизий чипов APU/PPU. Целью этого раздела является получение таймлайна появления пиратских APU/PPU, а также "каталогизирование" всех особенностей и глюков ревизий отдельных чипов.
